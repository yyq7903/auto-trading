回测/模拟盘数据：我们采集的 vs 平台实际的
一、核心价格数据
字段	我们采集的	平台实际的	是否一致
开盘价 (PTB)	Chainlink RPC getChainlinkData() 每秒读 0xfeaf968c	Polymarket 结算用 Chainlink BTC/USD Data Stream (https://data.chain.link/streams/btc-usd)	⚠️ 大部分一致，但有精度差
BTC 实时价格	Chainlink latestRoundData() → /10^8 = 76620.817	平台显示的 BTC 价格	✅ 同源
结算价 (btc_final)	未采集！ trades.jsonl 中 live 订单 btc_final=None	Polymarket 用 Chainlink Data Stream 在窗口结束时刻的价格结算	❌ 缺失
gap 计算	btc_entry - ptb	平台无 gap 字段，但 winner 由 btc_final vs ptb 决定	⚠️ 我们的 gap 是入场时的，不是结算时的

二、概率/价格数据
字段	我们采集的	平台实际的	是否一致

bestBid / bestAsk	CLOB WebSocket 推送	gamma API bestBid, bestAsk	✅ 一致

概率走势	未采集 — 没有逐秒存概率时间序列	gamma API 无历史概率；只有当前时刻的	❌ 缺失

四、结算结果

六、缺失字段（回测必须用到）
